## 📖 About

Welcome to my **PCB Design Portfolio**!

This repository showcases my work in **PCB Design**, **Embedded Systems**, and **Hardware Development** using **Altium Designer**. It includes complete PCB projects, custom schematic symbols, PCB footprints, integrated libraries, and 3D models developed for academic, personal, and real-world applications.

The primary goal of this repository is to document my hardware design journey while building a reusable library of electronic components and professionally designed PCB projects.
